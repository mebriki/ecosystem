# Algeria Ecosystem

A comprehensive directory of Algeria's innovation ecosystem. Discover startups, incubators, accelerators, coworking spaces, media resources, job portals, communities, and events that contribute to Algeria's tech and business landscape.

## Getting Started

### Run Project Locally

```sh
# Clone the repository using the project's Git URL.
git clone https://github.com/algeria-ecosystem/ecosystem.git

# Navigate to the project directory.
cd ecosystem

# Install the necessary dependencies.
npm i

# Start the development server with auto-reloading and an instant preview.
npm run dev
```
